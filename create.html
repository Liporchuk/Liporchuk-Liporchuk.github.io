Код сайта

Обычный код создания сайта выглядит следующим образом.

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название сайта!!!</title>
</head>
<body>
    <img src="icon.png" width="200px">
</body>
</html>
